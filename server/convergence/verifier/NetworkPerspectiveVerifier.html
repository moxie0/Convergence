This is a Convergence notary.

Convergence is a distributed system to replace traditional certificate authorities. If you connect to an SSL protected server, you are presented with a certificate that is intended to authenticate the server. Your Convergence client will transfer the url of the server and the certificate to several notaries. The notaries will then verify the authenticity.

This notary uses the NetworkPerspective verifier.

Given a pair of an url and a certificate, the notary will confirm the authenticity in the following cases:

  1. It has successfully verified the authenticity before and still has the result in its cache.
  2. The server presents the same certificate to the notary as it did to you.

Otherwise the notary will not confirm the authenticity.
